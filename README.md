# Order Cancellation Analysis Predictor

## Overview
This project is a machine learning web application developed using Flask and Scikit-learn to predict order cancellations based on customer and order-related information.

The application uses a trained machine learning model stored in Pickle (.pkl) format and provides predictions through a user-friendly web interface.

---

## Features
- Predicts order cancellations
- Flask-based web application
- Machine learning model integration
- Real-time prediction system
- Responsive frontend interface

---

## Technologies Used
- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- HTML/CSS
- JavaScript
- Pickle (.pkl)
